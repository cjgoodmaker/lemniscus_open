Built for privacy

Your health data never leaves your machine. No cloud, no HTTP server, no third-party access.

🔒

100% Local

All data is stored in a local SQLite database. The MCP server communicates with Claude Code over stdio -- nothing touches the network.

🔍

Semantic Search

Combines vector embeddings (384-dim MiniLM) with full-text keyword search and temporal decay ranking for accurate results.

📁

Drop-In File Support

Apple Health XML, Oura/Garmin JSON, PDFs, images, CSVs, and text files. Just copy them into data/ and index.

⚡

Fast Indexing

Processes 600MB Apple Health exports in ~2 minutes. Auto-aggregates millions of readings into queryable daily summaries.

🔄

Live Reindexing

Add new files while Claude Code is running. Ask Claude to reindex and your new data is searchable immediately.

🧩

MCP Native

Speaks the Model Context Protocol natively. No HTTP server, no REST API -- just a clean stdio interface that Claude Code understands.

Quick Start

Up and running in under 5 minutes. Requires macOS, Python 3.11+, and Claude Code.

Clone and setup

Clone the repo and run the setup script. It creates a virtual environment, installs dependencies, and downloads the embedding model (~90MB).

git clone https://github.com/cjgoodmaker/lemniscus_open.git && cd lemniscus_open && bash setup.sh

Create your account

Sign up and verify your email, then log in.

.venv/bin/python server.py signup

Add your health data

Copy Apple Health exports, lab reports, wearable data, clinical photos -- anything health-related into the data/ folder.

cp ~/Downloads/export.xml data/

Index and go

Index your data, then start Claude Code. The MCP server connects automatically.

.venv/bin/python server.py index && claude

How It Works

A local pipeline from raw files to natural language answers.

data/ (your files) ↓ Parsers — Apple Health XML, JSON, PDF, images, text ↓ Chunker — splits into searchable segments ↓ Embedder — 384-dim MiniLM vectors (runs locally via ONNX) ↓ SQLite — FTS5 full-text index + vector store ↓ MCP Server — stdio interface, no HTTP ↓ Claude Code — natural language queries over your health data
